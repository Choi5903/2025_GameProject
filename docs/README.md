# 🧩 기억 조각 모음 (Memory Piece Collection)

> **"데이터화 된 기억 속에 접속하여 진실을 파헤쳐라"** > PC 윈도우 UI 컨셉의 시간 역행 퍼즐 플랫포머 어드벤처

<br>

## 📌 프로젝트 개요 (Project Overview)

**[기억 조각 모음]**은 플레이어가 '기억 복원사'가 되어 의뢰인의 손상된 기억 데이터 속에 접속, 진실을 파헤치는 **2D 퍼즐 어드벤처 게임**입니다.  
친숙한 PC 바탕화면 UI를 게임의 인터페이스로 활용하며, **과거와 현재의 시간대를 오가며 인과율을 조작**하여 퍼즐을 해결하는 것이 핵심입니다.

* **개발 기간**: 2025.03 ~ 2025.06 (3학년 1학기)
* **개발 인원**: **1인 개발** (기획, 프로그래밍, 아트 전담)
* **개발 엔진**: Unity 2022 LTS
* **플랫폼**: PC (Windows)
* **장르**: 퍼즐 어드벤처 / 횡스크롤 플랫포머

<br>

## 🎮 핵심 시스템 (Key Features)

### 1️⃣ 메타 UI (Meta UI) 시스템
* **PC 데스크톱 환경 구현**: 게임 화면 전체가 하나의 PC 바탕화면처럼 구성되어 있습니다.
* **창(Window) 제어**: 인벤토리, 지도, 대화창 등이 윈도우 OS의 '창' 형태로 열리고 닫히며, 플레이어는 이를 자유롭게 배치하거나 최소화하며 멀티태스킹 하듯 게임을 진행합니다.

### 2️⃣ 시간대 변환 (Time Manipulation)
* **타임라인 스위칭**: 버튼 하나로 즉시 '과거'와 '현재'의 기억 데이터 레이어를 전환합니다.
* **인과율 퍼즐**: 과거 시간대에서 무너진 다리를 수리하면, 현재 시간대에서 지나갈 수 있게 되는 등 시간의 흐름을 이용해 퍼즐을 해결합니다.

### 3️⃣ 횡스크롤 탐험 & 픽셀 아트
* **사이버펑크 & 레트로 무드**: 데이터 세계를 표현하는 독특한 글리치 효과와 정교한 도트 그래픽.
* **플랫포머 액션**: 점프, 상호작용 등 직관적인 조작으로 기억 공간을 탐험합니다.
